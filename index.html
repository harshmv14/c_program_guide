Setup & First Program

Theory

To begin, you need a text editor to write source code and a compiler (like GCC) to translate it into machine-executable code. The basic structure of any C program involves including necessary libraries (like `stdio.h` for input/output), a `main` function which is the program's entry point, and a `return 0;` statement to indicate successful execution. The `printf()` function is your primary tool for displaying output to the console.

Practice Problems

1. Hello World:

Write a program that prints "Hello, World!" to the console.

#include <stdio.h>

int main(void) {
    printf("Hello, World!\\n");
    return 0;
}

Explanation: This program includes the standard input/output library, defines the main function, and uses `printf` to display the text. The `\n` creates a newline.

Variables & I/O

Theory

Variables are named storage locations for data. Key types include `int` (integers), `float`/`double` (decimal numbers), and `char` (single characters). `scanf()` is used to read user input. It is crucial to use the address-of operator `&` with `scanf` to tell it where to store the data in memory.

Practice Problems

1. Swap Values:

Declare two integer variables, swap their values using a third temporary variable, and print their values before and after.

#include <stdio.h>

int main(void) {
    int a = 10, b = 20, temp;
    printf("Before: a=%d, b=%d\n", a, b);
    temp = a;
    a = b;
    b = temp;
    printf("After: a=%d, b=%d\n", a, b);
    return 0;
}

Explanation: A `temp` variable is required to hold the value of `a` before it's overwritten, allowing the swap to complete without losing data.

Arrays

Theory

Arrays are fixed-size collections of elements of the same data type, stored in contiguous memory. They are accessed using a 0-based index. Careful index management is essential to avoid "out-of-bounds" errors, a common pitfall in C.

Practice Problems

1. Find Min/Max:

In an integer array, find and print the smallest and largest elements.

#include <stdio.h>

int main(void) {
    int data[] = {45, 12, 89, 5, 67, 23};
    int size = sizeof(data) / sizeof(data[0]);
    int min = data[0];
    int max = data[0];

    for (int i = 1; i < size; i++) {
        if (data[i] < min) min = data[i];
        if (data[i] > max) max = data[i];
    }
    printf("Min: %d, Max: %d\\n", min, max);
    return 0;
}

Explanation: Initialize `min` and `max` to the first element. Then, loop from the second element, comparing and updating `min` and `max` as needed.

Pointers

This is a core C concept. Pointers are variables that store memory addresses. They are essential for dynamic memory and efficient data manipulation.

Practice Problems

1. Swap with Pointers:

Write a function that takes pointers to two integers and swaps the values they point to. This demonstrates modifying caller data.

#include <stdio.h>

void swap(int *ptrA, int *ptrB) {
    int temp = *ptrA;
    *ptrA = *ptrB;
    *ptrB = temp;
}

int main(void) {
    int var1 = 5, var2 = 10;
    printf("Before: var1=%d, var2=%d\n", var1, var2);
    swap(&var1, &var2);
    printf("After: var1=%d, var2=%d\n", var1, var2);
    return 0;
}

Explanation: The `swap` function receives memory addresses. By dereferencing the pointers (`*ptrA`), it can modify the original variables in `main`, enabling a true swap.
